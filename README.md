## üíª Development

### Install Repository

```git
git clone https://github.com/HarshKumarraghav/SocialSeedlings.git
```

### Run Project

Install node modules and run the project.

```
npm install
or
yarn install
```

```
npm run dev
or
yarn run dev
```

### Finish

```
Your project is running on https://localhost:3000.
```
